Project Description : We aimed to develop a mobile-based system named Hospital Appointment and Queue Management System.
Our idea is to book hospital appointments online, select available time slots, receive appointment reminders and view real-time queue numbers.
We aimed to reduce long waiting times, manage overcrowded registration counters, prevents patients arriving too early or too late
, cope with no visibility of queue status and overcome inefficient appointment scheduling
